# 🛒 Super Store Sales Dashboard

## Dashboard Preview

![Sales Dashboard](Sales%20Dashboard%20Analysis.png)

## Project Overview

This dashboard analyzes Super Store sales performance by tracking revenue, profit, customer segments, shipping methods, payment modes, and regional performance. It enables business users to monitor KPIs, identify trends, and support data-driven decision-making.

### Key Features

* Sales & Profit KPIs
* Regional Sales Analysis
* Monthly Sales & Profit Trends (YoY)
* Sales by Customer Segment
* Sales by Ship Mode
* Sales by Payment Mode
* Product Category Analysis
* Interactive Regional Filters

### Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling

# 👥 HR Analytics Dashboard

## Dashboard Preview

![HR Dashboard](HR%20Dashboard%20Analysis.png)

## Project Overview

This HR Analytics Dashboard provides insights into employee attrition, workforce demographics, salary distribution, and departmental performance. It helps HR professionals understand attrition patterns and make informed workforce planning decisions.

### Key Features

* Employee Attrition Analysis
* Attrition by Job Role
* Attrition by Education
* Attrition by Salary
* Attrition by Age Group
* Department-wise Filtering
* HR KPI Cards

### Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling
